# 🚀 NASA Mission Explorer

## 🌌 Project Overview
NASA Mission Explorer is a web application that allows users to explore the Astronomy Picture of the Day (APOD) provided by NASA. Users can select any date to view space images or videos from that day, including their birthday.

---

## 🎯 Purpose
The purpose of this project is to:
- Learn how to work with public APIs
- Build an interactive and visually appealing UI
- Implement features like search, filtering, and data display

---

## 🌐 API Used
NASA APOD API  
https://api.nasa.gov/

Endpoint:
https://api.nasa.gov/planetary/apod

---

## ✨ Features

### ✅ Core Features
- Select a date to view APOD
- View title, description, and media (image/video)
- HD image toggle
- Full-screen image preview

### 🔄 Planned Features
- Favorites system (save images)
- Random date generator
- Search by date
- Filter/sort images
- Dark space-themed UI

---

## 🛠️ Technologies Used
- React (Vite)
- Tailwind CSS
- JavaScript (ES6+)
- NASA Open API
